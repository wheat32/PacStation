# PacStation

> [!WARNING]
> PacStation is currently in an early prerelease state.
> 
> Development is still in progress and the application is not yet ready for public testing or daily use. Features may be incomplete, unstable, or entirely nonfunctional, and package management operations may change significantly during development.

PacStation is a modern Qt-based graphical package manager for Arch Linux that unifies official repositories, AUR packages, and Flatpaks into a single desktop experience. Browse, install, update, and manage software through a fast native interface designed for Arch users who want the power of the terminal with the convenience of a polished GUI.
